# T&R Car Detail Website

A modern car detailing service website built with React, TypeScript, and Node.js.

## Tech Stack

**Frontend:**
- Vite + React 19 + TypeScript
- React Router for navigation
- Tailwind CSS for styling
- Framer Motion for animations
- React Icons for iconography

**Backend:**
- Node.js + Express + TypeScript
- SQLite for database
- Nodemailer for email notifications

## Getting Started

### Frontend

```bash
cd frontend
npm install
npm run dev
```

Frontend runs on http://localhost:3001 (or next available port)

### Backend

```bash
cd backend
npm install
npm run dev
```

Backend runs on http://localhost:5000

## Project Structure

- `frontend/` - React application
- `backend/` - Express API server

## Features

- Service pages with pricing
- Booking system
- Contact form
- FAQ section
- Responsive design
